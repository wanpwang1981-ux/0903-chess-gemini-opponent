# 中國暗棋 (Chinese Dark Chess)

這是一款使用 HTML, CSS, 和 JavaScript 建立的中國暗棋遊戲。玩家可以選擇與電腦對戰，或在同一裝置上進行雙人對戰。

## 遊戲規則

### 1. 棋盤與棋子
- **棋盤**: 遊戲在一個 4x8 的棋盤上進行。
- **棋子**: 使用中國象棋的 32 枚棋子，分為紅、黑兩方，每方 16 枚。
    - **紅方**: 帥(1), 仕(2), 相(2), 俥(2), 傌(2), 炮(2), 兵(5)
    - **黑方**: 將(1), 士(2), 象(2), 車(2), 馬(2), 包(2), 卒(5)

### 2. 遊戲設置
- 所有 32 枚棋子在洗勻後，面朝下（蓋著）隨機放置在棋盤的各個位置上。

### 3. 遊戲流程
- **開局**: 玩家可選擇「與電腦對戰」或「雙人對戰」。
- **決定顏色**: 由先手玩家任意翻開一枚棋子。該棋子的顏色即為先手玩家的顏色，另一方則執另一色。
- **回合**: 玩家輪流進行。在每個回合，玩家可以執行以下三種動作之一：
    1. **翻棋**: 翻開一枚面朝下的棋子。
    2. **移子**: 將己方面朝上的棋子，水平或垂直移動一格至相鄰的空位。
    3. **吃子**: 將己方面朝上的棋子，移動至被吃的敵方棋子位置上。吃子必須符合階級規則。

### 4. 吃子規則

#### 棋子階級 (由大到小)
帥(將) > 仕(士) > 相(象) > 俥(車) > 傌(馬) > 包(炮) > 兵(卒)

- **一般規則**: 高階級的棋子可以吃掉低階級的棋子。階級相同的棋子可以互吃。
- **特殊規則**:
    - **兵(卒)**: 雖然是階級最低的棋子，但可以吃掉對方最高階級的 **帥(將)**。
    - **炮(包)**:
        - **移動**: 作為移子時，只能走一格。
        - **吃子**: 必須跳過一個棋子（稱為「炮台」，無論顏色或明暗），在同一直線上吃掉遠方的任何敵方棋子。炮(包)與被吃的棋子之間必須有且僅有一個炮台。
    - **帥(將)**: 除了兵(卒)外，可以吃掉任何比自己階級低的敵方棋子。

### 5. 勝負判定
- **獲勝**:
    - 吃掉對方所有的棋子。
    - 對方無任何合法棋步可走（例如所有棋子都被吃光或被困住）。
- **和局**:
    - 雙方同意和局。
    - 出現重複循環的局面。
    - 連續指定步數內（例如 50 步）沒有任何吃子發生。
